Integration Steps

  1. Include the script: Add the chatbot-widget.js script to your page
  2. Initialize: Call window.ChatbotWidget.init() with your configuration
  3. Customize: Adjust colors, position, and messages to match your brand
  4. Test: Verify the widget appears and webhook responses work
⚠️ Note: Make sure your webhook endpoint is configured to handle POST requests and return the expected JSON format.

Troubleshooting

If the widget doesn't appear:

  • Check browser console for errors
  • Verify the script loads successfully (check Network tab)
  • Ensure window.ChatbotWidget.init is called after script loads
  • Test webhook endpoint separately
